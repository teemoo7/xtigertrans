xTigerTrans Viewer Application

This is a Firefox application. It illustrates the use of the "xtigertrans.js" library

  • Page : enter a file URL for an XTiger file to visualize (or use Page shortcut)
  • Page shortcut : select directly a predefined XTiger template file to visualize
  • View with : select a transformation to apply to generate the view

Press "Visualize" to render the XTiger template file with the selected transformation

NOTE: currently it supports only local file URL (i.e. URL starting with http: will cause a failure)

By default template files listes in the "Page shortcut" popup menu are stored in the "templates" folder.

By default transformations that appear in the right popup menu are stored as sub-folders within the "transformations" folder. Each transformation sub-folder must define the transformation in an "index.xhtml" file
