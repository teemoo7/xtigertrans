xTigerTrans Library Project

Home : http://xtigertrans.googlecode.com

Presentation

xTigerTrans is a (FireFox) Javascript library for client-side processing of XTiger template files.

This library has been developed in the framework of the PALETTE research project. Its initial purpose is to quickly prototype some template views to be implemented in template driven editing applications, such as DocReuse.

viewerApp ou comment visualiser vos templates XTiger ?

Pour visualiser vos propres templates XTiger, mettez les (fichiers finissant en .xtd) dans le répertoire "templates", puis ouvrez le fichier "viewerApp/viewer.xhtml" dans FireFox. L'application construit un menu "Page shortcut" au démarrage pour sélectionnner tous les templates qui se trouvent dans le répertoire "templates".

De même vous pouvez créer vos propres transformations (fichier index.xhtml) et les placer dans un sous-répertoire dans le répertoire "transformations", lu au démarrage.

The library comes with a viewerApp application. You can try it to see some examples of template visualizations. Launch it by opening the "viewerApp/viewer.xhtml" into Firefox.

Content of the package

README.html      : this file
docs/            : documentation files (currently Jonathan project reports)
templates/       : default location for XTiger template files to visualize
transformations/ : default location for transformations to apply to visualize templates, 
                   each sub-folder must define the transformation in an "index.xhtml" file
viewerApp/       : sample application to apply tranformations to templates (viewer.xhtml)
xtiger/          : library Javascript code and utilities
